# Product

## Register

product

## Users

ทีม Botnoi internal — ใช้ภายในองค์กรสำหรับจัดการบทสนทนา LINE OA ติดตาม task และดูแลทีม

## Product Purpose

LINE OA CRM dashboard สำหรับ Botnoi ช่วยให้ทีมเห็น conversation ลูกค้า มอบหมายงาน และจัดการทีมได้จากที่เดียว

## Brand Personality

Reliable, Precise, Internal-grade — ไม่ต้องขาย ต้องใช้งานได้จริงทุกวัน

## Anti-references

- ห้ามดู "colorful startup dashboard" ที่มี gradient ทุกปุ่มและ chart ทุก widget
- ห้ามดู Notion-like monochrome ที่ไม่มี visual hierarchy
- ห้ามดู consumer app ที่ใช้ animation เยอะเกิน

## Design Principles

1. **Content first** — UI ให้ทางกับ data ไม่ใช่แย่งความสนใจ
2. **Minimal chrome** — ตัด decoration ที่ไม่ทำให้ workflow เร็วขึ้น
3. **Consistent density** — spacing สม่ำเสมอ อ่านง่าย ไม่ต้อง scan นาน
4. **Botnoi blue as anchor** — ใช้สีฟ้าเป็น primary action เท่านั้น ส่วนที่เหลือเป็น neutral

## Accessibility & Inclusion

WCAG AA — contrast ≥4.5:1 สำหรับ body text, ≥3:1 สำหรับ large/bold
